# Currency Converter (Конвертер валют)

**Автор:** : Жамборова Дарина
**Вариант:** Currency Converter (Конвертер валют)


## Описание программы

Программа конвертирует деньги из одной валюты в другую по актуальному курсу.  
Используется бесплатный API (Frankfurter), для работы нужен интернет.

Курсы обновляются каждый день. API бесплатный, регистрация не нужна.

## Где взять API

В программе используется **Frankfurter API** — он бесплатный и не требует ключа.  
Ничего регистрировать не нужно, всё работает сразу.

## Что нужно для запуска

- Python 3
- tkinter, json, requests, datetime
